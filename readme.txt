<구현한 내용>
• 화면에 바닥을 그린다.
• 원근투영을 적용한다.
• 사용자에게 가로와 세로의 나누기 개수를 입력 받는다.
• 콘솔창에서 가로와 세로의 개수를 입력 받도록 한다.
• 나누기 개수는 5 ≤ 개수 ≤ 25으로 한다. 
• 입력 받은 개수 만큼으로 가로와 세로를 나누고 각 칸에 육면체를 그린다.
• 화면의 한 부분에 탑뷰의 미니맵이 그려진다.
1: 애니메이션 1 : 육면체들이 다른 속도 다른 높이값으로 다양하게 움직이다.
• t: 조명을 켠다/끈다.
• c: 조명 색을 바꾼다. 4개의 색 적용
• y/Y: 카메라가 바닥의 y축을 기준으로 양/음 방향으로 회전한다. (공전, 다시 누르면 회전을 중지한다.)
• +/-: 육면체 이동하는 속도 증가/감소
• r: 모든 값 초기화 (새롭게 가로세로 값을 입력받아 애니메이션 시작한다)
• q: 프로그램 종료
• *** 키보드 명령어를 콘솔창에 출력한다.


<구현하지 못한 내용>

• y/Y: 카메라가 바닥의 y축을 기준으로 양/음 방향으로 회전한다. (공전, 다시 누르면 회전을 중지한다.)
• +/-: 육면체 이동하는 속도 증가/감소
• r: 모든 값 초기화 (새롭게 가로세로 값을 입력받아 애니메이션 시작한다)
• q: 프로그램 종료
