<구현한 내용>
1. 화면에 바닥을 그린다.
2. 원근투영을 적용한다.
3. 사용자에게 가로와 세로의 나누기 개수를 입력 받는다.
	-콘솔창에서 가로와 세로의 개수를 입력 받도록 한다.
	-나누기 개수는 5 ≤ 개수 ≤ 25으로 한다. 

4. 입력 받은 개수 만큼으로 가로와 세로를 나누고 각 칸에 육면체를 그린다.
5. 화면의 한 부분에 탑뷰의 미니맵이 그려진다.
6. 세 종류의 다른 애니메이션을 적용한다.
	애니메이션 1 : 육면체들이 다른 속도 다른 높이값으로 다양하게 움직인다.
	애니메이션 2  : 육면체들이 파도와 같이 움직인다.
	애니메이션 3  : 짝수번 행의 육면체들과 홀수번 행의 육면체들이 각각 다르게 상하로 움직인다.

7. 화면 위쪽에 조명을 넣는다.
	조명을 켠다/끈다.
	조명 색을 바꾼다. 4개의 색 적용

8. 카메라가 바닥의 y축을 기준으로 양/음 방향으로 회전한다. (공전, 다시 누르면 회전을 중지한다.)

9. 육면체 이동하는 속도 증가/감소
10. 모든 값 초기화 (새롭게 가로세로 값을 입력받아 애니메이션 시작한다)
11. 프로그램 종료
12. 키보드 명령어를 콘솔창에 출력한다.
13. 육면체가 나타나는 애니메이션
	-위에서 육면체가 떨어진다.

<키보드 명령어>
• 1: 애니메이션 1
• 2: 애니메이션 2
• 3: 애니메이션 3
• t: 조명을 켠다/끈다.
• c: 조명 색을 바꾼다. 4개의 색 적용
• y/Y: 카메라가 바닥의 y축을 기준으로 양/음 방향으로 회전한다. (공전, 다시 누르면 회전을 중지한다.)
• +/-: 육면체 이동하는 속도 증가/감소
• r: 모든 값 초기화 (새롭게 가로세로 값을 입력받아 애니메이션 시작한다)
• 7: 1인칭 카메라 
• 9: 3인칭 카메라 
• p: 키보드 명령어를 콘솔창에 출력
• o/O: 직각 투영/ 원근 투영
• q: 프로그램 종료
